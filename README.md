# react-starter-project

[![js-standard-style](https://cdn.rawgit.com/feross/standard/master/badge.svg)](http://standardjs.com)

## 安装

```
yarn
```

## 启动

```
yarn run serve
```

## 测试地址

开发环境

http://localhost:8080/assets/

生产环境（验证用）

http://localhost:8080/assets-dist/

## 打包

```
yarn run build
```

## 上线发布

assets-dist 目录打包上传到服务器对应目录即可
